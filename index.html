Wimmy

A minimal .NET application for Windows. Much like Launchy, it runs in the background. Once invoked with a keyboard shortcut, it displays a list of running applications and allows typing in the name of the application to switch to.

This is essentially an Alt-Tab alternative for when you have too many open Windows, or when it's quicker to switch to a certain window (e.g. Invoke Wimmy, type "cmd" to switch to terminal).
